# Installation

Right now, the best way to get Swirl is to build it from source. 

## Prerequisites

Before you start, make sure you have the following tools installed:
- Git
- C++ compiler with C++23 support (e.g., `g++` or `clang++` or `msvc`)
- CMake (version 3.20 or newer)
- LLVM Development libraries (version 17 or newer)

## Linux
<Tabs groupId="os">
<TabItem value="Ubuntu" label="Ubuntu">
First, update your packages:
```bash
sudo apt update && sudo apt upgrade -y
```
Then, install the required packages:
```bash
sudo apt install git cmake clang llvm-dev liblld-dev
```
</TabItem>

<TabItem value="fedora" label="Fedora">
First, update your packages:
```bash
sudo dnf update -y
```
Then, install the required packages:
```bash
sudo dnf install git cmake clang llvm-devel lld-devel
```
</TabItem>
<TabItem value="arch" label="Arch Linux">
First, update your packages:
```bash
sudo pacman -Syu
```
Then, install the required packages:
```bash
sudo pacman -S git cmake clang llvm lld
```
</TabItem>
</Tabs>

## Windows

- First download MSYS from [msys2.org](https://www.msys2.org/).

- Open the CLANG64 shell.

- Update packages:
    ```bash
    pacman -Syu
    ```

- Install the required packages:
    ```bash
    pacman -S mingw-w64-clang-x86_64-cmake mingw-w64-clang-x86_64-toolchain
    ```
        - It will install cmake, clang, llvm dev libs, git, make etc.

- Set up the environment variables:
    Add the following line to your path environment variable in windows settings
    ```
    C:\msys64\clang64\bin
    ```
    [Here's](https://www.computerhope.com/issues/ch000549.htm#windows11) a guide for that.

Your system should now be set up to build Swirl.

## Building
First clone the repository and navigate into it:
```bash
git clone https://github.com/swirllang/swirl.git
```
```bash
cd swirl
```

Generate the build files:
```bash
cmake -B build -S .
```

Build the project:
```bash
cmake --build build --config Release
```
The Swirl binary will be in the `build` directory.

## Running
To run Swirl, you can use the following command:
```bash
./build/swirl
```
